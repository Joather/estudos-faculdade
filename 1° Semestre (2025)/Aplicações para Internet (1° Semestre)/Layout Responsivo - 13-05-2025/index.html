Bootstrap O que é?

O Bootstrap é um framework front-end gratuito e de código aberto, desenvolvido inicialmente pelo Twitter em 2011. Ele fornece uma coleção de ferramentas (HTML, CSS e JavaScript) para facilitar a criação de sites e aplicativos web responsivos e visualmente consistentes.
